CSCE 242: Web Applications

Ashiya Branch

CSCE 242 will teach students how to use HTML and CSS. These tools will then help design a website. Students will learn JavaScriprt for functionality. Node a JavaScript to store and retrieve data from a server.

Assignments

  • Assignment 1- Basic HTML

In assignment 1 we needed to create a basic html page using properly formatted html code. The code needs to include h1, p, 2 images, bulleted list, a table, and 2 external links.

  • Assignment 2- Basic CSS

In assignment 2 we create a new page like assignment 1 but we add basic css to the page. We also add css to our main page.

  • Assignment 3- Page Layout

In assignment 3 we create a page that follows a flexbox layout and use media query to help switch the page between horizontal and vertical view.

  • Assignment 4- Main Page CSS

In assignment 4 we change the layout of the main page to a flexbox layout.
